# hotpotato
A tool for testing Ravencoin

Make sure you have pip or pip3  
```pip install tinydb```  
```pip install python-bitcoinrpc```  

Make sure you've added these to your raven.conf file (not in testnet folder)
```
rpcuser=rpcuser
rpcpassword=rpcpass555
```

Run the qt with  
```-testnet -server```

```python hotpotato.py```  
```python fission.py```
